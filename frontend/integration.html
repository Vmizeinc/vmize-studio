Shopify Installation

Add VMIZE virtual try-on to your Shopify store in 5 minutes

Quick Overview

Installing VMIZE on Shopify is simple and requires no coding. You'll add a small code snippet to your product page template.

‚è±Ô∏è Installation Time: 5-10 minutes
üë®‚Äçüíª Technical Skill: No coding required
üîß Requirements: Shopify plan (any tier), VMIZE API key

Step-by-Step Installation

1

Get Your API Key

First, you need your VMIZE API key:

  1. Sign up at VMIZE Studio
  2. Go to your dashboard
  3. Navigate to Settings ‚Üí API Keys
  4. Copy your Live API key (starts with vmize_pk_live_)
‚ö†Ô∏è Important: Keep your API key secure. Never share it publicly or commit it to version control.
2

Access Your Shopify Theme Editor

  1. Log in to your Shopify admin panel
  2. Go to Online Store ‚Üí Themes
  3. Find your active theme and click Actions ‚Üí Edit code
3

Add VMIZE Script to Product Template

Locate your product template file:

  1. In the left sidebar, navigate to Sections
  2. Find and open main-product.liquid or product-template.liquid
  3. Find the section where the "Add to Cart" button is located
  4. Add this code just before the closing tag:
<!-- VMIZE Virtual Try-On Widget -->
<div id="vmize-tryon-container"></div>
<script src="https://cdn.vmizestudio.com/widget.js"></script>
<script>
  VmizeWidget.init({
    apiKey: 'YOUR_API_KEY_HERE',
    productId: '{{ product.id }}',
    productImage: '{{ product.featured_image | img_url: 'master' }}',
    buttonText: 'Try It On',
    position: 'before-add-to-cart'
  });
</script>

Replace YOUR_API_KEY_HERE with your actual VMIZE API key.

4

Save and Test

  1. Click Save in the theme editor
  2. Visit any product page on your store
  3. You should see the "Try It On" button
  4. Click it to test the virtual try-on feature
‚úÖ Success! Your VMIZE virtual try-on is now live on your Shopify store.

Troubleshooting

Button Not Showing

  • Verify your API key is correct and active
  • Check browser console for JavaScript errors
  • Ensure the widget script is loading
  • Make sure you saved the theme changes

Try-On Not Working

  • Verify you have available credits
  • Check that product images are accessible
  • Ensure images are in supported formats
  • Try clearing your browser cache

WordPress Installation

Add VMIZE virtual try-on to your WordPress site

Quick Overview

You can integrate VMIZE into WordPress using either a plugin or manual code insertion.

üë®‚Äçüíª Technical Skill: Basic WordPress knowledge
üîß Requirements: WordPress 5.0+, VMIZE API key

Method 1: Using Code Snippets Plugin (Recommended)

1

Install Code Snippets Plugin

  3. Install and activate the plugin
2

  1. Go to Snippets ‚Üí Add New
  2. Title: "VMIZE Virtual Try-On"
  3. Paste this code:
function vmize_add_widget() {
    if (is_product() || is_page()) {
        ?>

        <script>

Custom Website Installation

Add VMIZE to any custom e-commerce site

Quick Overview

For custom websites, use either the JavaScript widget or direct API integration.

‚è±Ô∏è Installation Time: 15-30 minutes
üë®‚Äçüíª Technical Skill: JavaScript/HTML knowledge
üîß Requirements: Custom website, VMIZE API key

Method 1: JavaScript Widget (Easiest)

1

Add Widget Script

<!-- Place before </body> -->
<script src="https://cdn.vmizestudio.com/widget.js"></script>
2

Initialize Widget

<div id="vmize-tryon-container"></div>

<script>
  VmizeWidget.init({
    apiKey: 'YOUR_API_KEY_HERE',
    productId: 'product-123',
    productImage: 'https://yoursite.com/product.jpg',
    buttonText: 'Try It On',
    buttonColor: '#6B4CE6'
  });
</script>

Method 2: Direct API Integration

1

Create Try-On Request

const createTryOn = async (modelImage, garmentImage) => {
  const response = await fetch('https://api.vmizestudio.com/v1/tryon/generate', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'x-api-key': 'YOUR_API_KEY_HERE'
    },
    body: JSON.stringify({
      model_image: modelImage,
      garment_image: garmentImage,
      category: 'tops'
    })
  });
  
  const data = await response.json();
  return data.prediction_id;
};
2

Poll for Results

const getTryOnResult = async (predictionId) => {
  const response = await fetch(
    `https://api.vmizestudio.com/v1/tryon/${predictionId}`,
    { headers: { 'x-api-key': 'YOUR_API_KEY_HERE' } }
  );
  
  const data = await response.json();
  
  if (data.status === 'completed') {
    return data.output.image_url;
  }
  
  await new Promise(resolve => setTimeout(resolve, 2000));
  return getTryOnResult(predictionId);
};
