Introduction

VMIZE Studio provides AI-powered virtual try-on technology for e-commerce. Our API handles authentication, usage tracking, and billing with a simple integration.

Quick Start

1. Get Your API Key

After signing up, you'll receive your API key:

2. Generate Try-On

const response = await fetch('https://api.vmizestudio.com/api/tryon/generate', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'vmize_pk_live_yourkey'
  },
  body: JSON.stringify({
    model_image: 'data:image/jpeg;base64,...',
    garment_image: 'https://yourstore.com/product.jpg',
    category: 'tops',
    mode: 'quality',
    garment_photo_type: 'flat-lay',
    num_samples: 1
  })
});
const data = await response.json();
console.log(data.prediction_id);

3. Poll for Results

const checkResult = async (predictionId) => {
  const response = await fetch(
    \`https://api.vmizestudio.com/api/tryon/\${predictionId}\`,
    { headers: { 'x-api-key': 'vmize_pk_live_yourkey' } }
  );
  const data = await response.json();
  
  if (data.status === 'completed') {
    return data.output[0]; // Result image URL
  } else if (data.status === 'failed') {
    throw new Error(data.error?.message);
  }
  // Still processing - wait and retry
  await new Promise(r => setTimeout(r, 2000));
  return checkResult(predictionId);
};

Best Practices

Image Quality

  • Use images 512-2000px resolution
  • Ensure good lighting and clear visibility
  • Compress to 500KB-2MB before upload
  • Use plain backgrounds for garments

Security

  • Never expose API keys in frontend code
  • Call VMIZE API from your backend only
  • Use environment variables for keys
